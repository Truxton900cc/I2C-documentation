# &#x09;	I2C



* #### Resistors



A 100KHz, dos Resistencias 14k ohms



* RB0 y RB1, Deben ser configurados como 1 en el TRIS.



* REGISTROS

  1. SSPCON1 (CONTROL)
  2. SSPCON2 (CONTROL)
  3. SSPSTAT (STATUS)
  4. SSPBUF  (TX/RX)
  5. SSPSR   (SHIFT REGISTER)
  6. SSPADD  (ADDRESS REGISTER)
  7. SSPIF   (INTERRUPCION)



SSPCON1, SSPCON2, SSPSTAT = CONTROL, CONTROL, STATUS.



SSPCON1, SSPCON2 SE PUEDE ESCRIBIR Y LEER.



SSPSTAT, LOS BITS 7 Y 6 SE PUEDEN ESCRIBIR Y LEER.



SSPBUF, ES DONDE SE LEEN O ESCRIBEN DATOS.



SSPADD:

* SLAVE MODE:  ESTE REGISTRO SE USA PARA GUARDAR LA DIRECCION DE SI MISMO
* MASTER MODE: LOS 7BITS BAJOS,SE COMPORTA COMO UN BAUD RATE GENERATOR



##### **RX:**

CUANDO SE RECIVE UN DATO, SE GUARDA EN **SSPBUF** Y SE ACTIVA LA INTERRUPCION **SSPIF**



##### **TX:**

CUANDO SE TRANSMITE UN DATO, SE UTILIZA **SSPBUF** Y NO HAY INTERRUPCION.



**OPERATION:**



* SSPADD: ES UN REGISTRO QUE CARGA EL VALOR QUE DETERMINA LA FRECUENCIA DE RELOJ.	

  * **ACLARACION**:
  * EL MAESTRO ES QUIEN MARCA EL RITMO DE LA CONVERSACION.
  * LA FORMULA DE SSPADD = (FOSC / 4(FSCL)) - 1



* **SSPBUF:** SE UTILIZA **PRIMERO** PARA LA DIRECION DE ESCLAVO

  * ACLARACION:
  * LOS ESCLAVOS TIENEN 2 DIRECCIONES, UNA PARA ENVIAR  OTRA PARA RECIBIR.
  * CUALQUIERA DE LAS DOS CABE EN 7bits.
  * 0 = MAESTRO ESCRIBE
  * 1 = MAESTRO LEE
